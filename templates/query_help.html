Type a plain-English question about your telemetry data. The AI generates a ClickHouse SQL query, executes it, and displays the results.

Query Options
Option Description
Execute query When enabled (default), the generated SQL is executed and results are returned. Disable to see only the generated SQL without running it.
Render chart When enabled, the AI also generates an ECharts configuration to visualize the query results as a chart.
Example Questions
  • How many log entries were recorded per service in the last hour?
  • What are the top 10 slowest trace spans in the last 24 hours?
  • Show me the error rate per service over the past week.
  • Which services have the most unresolved errors?
  • List all RUM sessions longer than 5 minutes from today.
Beta Limitations
The Query feature is beta. Generated SQL may occasionally be incorrect or use unsupported ClickHouse syntax.
  • If a query fails, try rephrasing your question or simplify the request.
  • The AI only has read access to the embedded chDB database — it cannot modify data.
  • Complex joins across many tables may produce slow or empty results.
  • Enable SOBS_ENABLE_QUERY=1 in environment variables to activate this feature.
